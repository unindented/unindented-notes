- URL :: https://ladybird.org/

Web browser backed by a 501(c)(3) non-profit.

- Not based on Chromium (unlike Chrome, Edge, Brave, Arc, Opera, [[id:b883de17-f6ed-4eda-9f4d-c336fbd7789f][Vivaldi]]).
- Not trying to monetize, so no search deals, promoted bookmarks, or similar crap.

They're targeting a first alpha release for 2026.
